mysql 设置远程访问

  1. sudo vim /etc/mysql/mysql.conf.d/mysqld.cnf
  2. 注释掉 ‘#bind-address = 127.0.0.1’
  3. 进入 mysql
  4. GRANT ALL PRIVILEGES ON . TO ‘username’@'%’ IDENTIFIED BY ‘password’ WITH GRANT OPTION;
  5. flush PRIVILEGES
  6. 退出并重启 mysql
